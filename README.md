## ⚙️ How GitHub Sync works

NeetCode's sync is set-and-forget. Here's the flow:

```
You solve a problem on NeetCode.io
            │
            ▼
    Solution accepted ✅
            │
            ▼
  Auto-committed to this repo 🔄
            │
            ▼
   Organized by topic + problem ID 📁
```

| Feature | What it does |
|---|---|
| 🔗 **Connect** | Link GitHub at [neetcode.io/profile/github](https://neetcode.io/profile/github) |
| ⚡ **Auto-commit** | Every submission → instant push (configurable) |
| 📦 **Bulk Sync** | Push all past solutions at once |
| 🧹 **Manual Sync** | Sync or remove individual submissions from problem page |
